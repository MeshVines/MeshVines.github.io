Features

Wine Fermentation Model

The Boulton model (Boulton, 1979) describes the kinetics of juice to wine based on commonly used wine yeast strains of Saccharomyces Cerevisiae and has been validated on many commercial fermentations across nearly all winemaking conditions.

Fermentation Release Rates

The current and future energy (as heat) and carbon dioxide release rates are calculated based on the fermentation rate and stoichemetric equation of yeast metabolism of sugar. This information can be used to manage refrigeration loads.

Detect Problem Fermentations

MeshVines API helps winemakers diagnose stuck or sluggish fermentations while there is still time to intervene. MeshVines API can also be used to predict fermentation outcomes from process changes such as a temperature change, nitrogen addition or reinoculation.

Easy Integration

As a REST API, MeshVines API is easy to integrate into other modern software. See the documentation for details on API endpoints and JSON message formats.

Fast

Built in the cloud, MeshVines API spends no more than 3 seconds matching the measured data to the Boulton model using a novel nonlinear parameter estimation algorithm. This makes applications built with MeshVines API fast and responsive.

Flexible Units

Use MeshVines API's built-in conversion functions to enable commonly used winemaking units such as Brix, Specific Gravity, Baume, Oeschle and Kg/m3
